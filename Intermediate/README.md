
## **Intermediate Level (Routing & VLANs)**

> Focus: VLANs, inter-VLAN routing, static & dynamic routing, ACLs, and small enterprise scenarios.

1. **01_InterVLAN_Routing_RouterOnAStick** – Configure multiple VLANs and inter-VLAN routing.
    
2. **02_Static_Routing_MultiRouter** – Connect 2-3 routers, configure static routes, test connectivity.
    
3. **03_RIP_Routing_Basics** – Implement RIP, verify routing tables, test network reachability.
    
4. **04_OSPF_Basic_Configuration** – Configure OSPF, multiple areas, verify neighbor relationships.
    
5. **05_ACL_Basics_IP_Filtering** – Implement standard and extended ACLs to block/allow traffic.
    
6. **06_DHCP_Server_and_Client_Subnets** – Configure DHCP server and client across multiple subnets.
    
7. **07_NAT_PAT_Basic_Labs** – Configure NAT and PAT to simulate internet access.
    
8. **08_VLAN_Trunking_and_VTP** – Implement VLAN trunking, VTP configuration, and switch management.
    
9. **09_Static_and_Default_Route_Combos** – Mix static routes with default routes for branch networks.
    
10. **10_Basic_Wireless_LAN_Setup** – Configure a wireless network, SSID, WPA2, connect PCs
